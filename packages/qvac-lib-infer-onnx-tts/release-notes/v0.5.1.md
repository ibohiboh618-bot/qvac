# QVAC TTS ONNX Addon v0.5.1 Release Notes

This release focuses on documentation and cleanup. The README now covers both the Chatterbox and Supertonic TTS engines with full usage guides, API references, and a side-by-side comparison. Legacy Piper and eSpeak-ng references have been removed from the benchmark suite, and the `@qvac/dl-hyperdrive` dependency has been dropped.
