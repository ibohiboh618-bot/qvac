```ts
function heartbeat(params?: { delegate?: { healthCheckTimeout?: number; providerPublicKey: string; timeout?: number; topic: string } }): Promise<HeartbeatResponse>;
```

Sends a heartbeat round-trip to verify that a delegated provider is online or that the local SDK worker is responsive. When called without arguments, checks the local worker.

## Parameters

| Name | Type | Required? | Description |
| --- | --- | :---: | --- |
| params | `\{ delegate?: \{ healthCheckTimeout?: number; providerPublicKey: string; timeout?: number; topic: string \} \}` | ✗ | Delegation target to check |

## Returns

`Promise<HeartbeatResponse>`

| Field | Type | Description |
| --- | --- | --- |
| number | `number` | — |
| type | `"heartbeat"` | — |

## Throws

| Error | When |
| --- | --- |
| `QvacErrorBase` | When the provider is unreachable or the response is invalid. |
| `INVALID_RESPONSE_TYPE` | Response type does not match expected `"heartbeat"` |

## Examples

```typescript
// Check if a delegated provider is online
try {
  await heartbeat({
    delegate: { topic: "topicHex", providerPublicKey: "peerHex", timeout: 3000 },
  });
  console.log("Provider is online");
} catch {
  console.log("Provider is offline");
}
```

```typescript
// Check if the local SDK worker is responsive
await heartbeat();
```
