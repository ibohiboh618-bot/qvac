---
title: "modelRegistrySearch( )"
titleStyle: code
description: Searches the model registry with optional filters for model type, engine, and quantization.
---

```ts
function modelRegistrySearch(params: ModelRegistrySearchParams): Promise<ModelRegistryEntry[]>;
```

## Parameters

| Name | Type | Required? | Default | Description |
| --- | --- | :---: | --- | --- |
| params | [`ModelRegistrySearchParams`](#modelregistrysearchparams) | ✓ | `\{\}` | Search filters (all optional). |

### `ModelRegistrySearchParams`

| Field | Type | Required? | Description |
| --- | --- | :---: | --- |
| filter | `string` | ✗ | Free-text filter on model name |
| engine | `string` | ✗ | Filter by inference engine |
| quantization | `string` | ✗ | Filter by quantization level |
| modelType | `"llm" \| "whisper" \| "embeddings" \| "nmt" \| "parakeet" \| "tts" \| "ocr" \| "diffusion" \| "vad" \| "other"` | ✗ | Filter by model type (alias for `addon`) |
| addon | `"llm" \| "whisper" \| "embeddings" \| "nmt" \| "parakeet" \| "tts" \| "ocr" \| "diffusion" \| "vad" \| "other"` | ✗ | Filter by addon type |

## Returns

`Promise<`[`ModelRegistryEntry[]`](../modelRegistryGetModel#modelregistryentry)`>` — Matching model entries.

## Throws

| Error | When |
| --- | --- |
| `ModelRegistryQueryFailedError` | When the registry query fails. |
| `QVAC_MODEL_REGISTRY_QUERY_FAILED` | The registry query fails |

## Example

```typescript
// Search LLM models
const llmModels = await modelRegistrySearch({ modelType: "llm" });

// Search by name
const llamaModels = await modelRegistrySearch({ filter: "llama" });

// Combined filters
const models = await modelRegistrySearch({
  modelType: "llm",
  quantization: "Q4_K_M",
});
```
