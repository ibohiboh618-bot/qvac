---
title: "ocr( )"
titleStyle: code
description: Performs Optical Character Recognition (OCR) on an image to extract text.
---

```ts
function ocr(params: OCRClientParams): { blocks: Promise<{ bbox?: [number, number, number, number]; confidence?: number; text: string }[]>; blockStream: AsyncGenerator<{ bbox?: [number, number, number, number]; confidence?: number; text: string }[]>; stats: Promise<{ detectionTime?: number; recognitionTime?: number; totalTime?: number } | undefined> };
```

## Parameters

| Name | Type | Required? | Description |
| --- | --- | :---: | --- |
| params | [`OCRClientParams`](#ocrclientparams) | ✓ | The OCR parameters |

### `OCRClientParams`

| Field | Type | Required? | Description |
| --- | --- | :---: | --- |
| modelId | `string` | ✓ | The identifier of the loaded OCR model |
| image | `string \| Buffer<ArrayBufferLike>` | ✓ | Image input as file path (string) or image buffer |
| options | `\{ paragraph?: boolean \| undefined; \}` | ✗ | Optional OCR options |
| stream | `boolean` | ✗ | Whether to stream blocks as they're detected |

## Returns

`object` — Object with the following fields:

| Field | Type | Description |
| --- | --- | --- |
| blocks | `Promise<\{ bbox?: [number, number, number, number]; confidence?: number; text: string \}[]>` | All detected text blocks (populated when `stream: false`) |
| blockStream | `AsyncGenerator<\{ bbox?: [number, number, number, number]; confidence?: number; text: string \}[]>` | Stream of detected text blocks (active when `stream: true`) |
| stats | `Promise<\{ detectionTime?: number; recognitionTime?: number; totalTime?: number \} \| undefined>` | Performance statistics |

### `blocks`

| Field | Type | Required? | Description |
| --- | --- | :---: | --- |
| bbox | `[number, number, number, number]` | ✗ | — |
| confidence | `number` | ✗ | — |
| text | `string` | ✓ | — |

### `stats`

| Field | Type | Required? | Description |
| --- | --- | :---: | --- |
| detectionTime | `number` | ✗ | — |
| recognitionTime | `number` | ✗ | — |
| totalTime | `number` | ✗ | Total embedding time in milliseconds |

## Example

```typescript
// Non-streaming mode (default) - get all blocks at once
const { blocks } = ocr({ modelId, image: "/path/to/image.png" });
for (const block of await blocks) {
  console.log(block.text, block.bbox, block.confidence);
}

// Streaming mode - process blocks as they arrive
const { blockStream } = ocr({ modelId, image: imageBuffer, stream: true });
for await (const blocks of blockStream) {
  console.log("Detected:", blocks);
}
```
