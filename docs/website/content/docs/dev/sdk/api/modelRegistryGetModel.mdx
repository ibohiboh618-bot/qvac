---
title: "modelRegistryGetModel( )"
titleStyle: code
description: Fetches a single model entry from the registry by its path and source.
---

```ts
function modelRegistryGetModel(registryPath: string, registrySource: string): Promise<ModelRegistryEntry>;
```

## Parameters

| Name | Type | Required? | Description |
| --- | --- | :---: | --- |
| registryPath | `string` | ✓ | Registry-relative path of the model to fetch. |
| registrySource | `string` | ✓ | Registry source identifier (e.g., `"huggingface"`, `"local"`). |

## Returns

`Promise<`[`ModelRegistryEntry`](../shared-types#modelregistryentry)`>` — The matching model entry.

| Field | Type | Description |
| --- | --- | --- |
| addon | `"llm" \| "whisper" \| "embeddings" \| "nmt" \| "vad" \| "tts" \| "ocr" \| "parakeet" \| "diffusion" \| "other"` | — |
| blobBlockLength | `number` | — |
| blobBlockOffset | `number` | — |
| blobByteOffset | `number` | — |
| blobCoreKey | `string` | — |
| engine | `"sdcpp-generation" \| "llamacpp-embedding" \| "llamacpp-completion" \| "nmtcpp-translation" \| "onnx-ocr" \| "parakeet-transcription" \| "onnx-tts" \| "whispercpp-transcription" \| "onnx-vad"` | — |
| expectedSize | `number` | — |
| modelId | `string` | — |
| name | `string` | — |
| params | `string` | — |
| quantization | `string` | — |
| registryPath | `string` | — |
| registrySource | `string` | — |
| sha256Checksum | `string` | — |

## Throws

| Error | When |
| --- | --- |
| `ModelRegistryQueryFailedError` | When the model cannot be located or the registry query fails. |
| `QVAC_MODEL_REGISTRY_QUERY_FAILED` | The registry query fails or model is not found |

## Example

```typescript
const model = await modelRegistryGetModel("llama-3.2-3b-q4", "qvac");
console.log(model.name, model.expectedSize);
```
