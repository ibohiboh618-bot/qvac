---
title: "ragDeleteEmbeddings( )"
titleStyle: code
description: Deletes document embeddings from the RAG vector database.
---

```ts
function ragDeleteEmbeddings(params: RagDeleteEmbeddingsParams, options?: RPCOptions): Promise<void>;
```

## Parameters

| Name | Type | Required? | Description |
| --- | --- | :---: | --- |
| params | `RagDeleteEmbeddingsParams` | ✓ | The parameters for deleting embeddings |
| options | `RPCOptions` | ✗ | Optional RPC options (timeout, profiling, force new connection, etc.). |

### `params`

| Field | Type | Required? | Description |
| --- | --- | :---: | --- |
| ids | `string[]` | ✓ | Array of document IDs to delete |
| modelId | `string` | ✗ | — |
| workspace | `string` | ✗ | Workspace to delete from (default: "default") |

### `options`

| Field | Type | Required? | Description |
| --- | --- | :---: | --- |
| forceNewConnection | `boolean` | ✗ | When `true`, skip any cached RPC connection and open a fresh one for this call. |
| healthCheckTimeout | `number` | ✗ | Timeout in milliseconds for the health-check probe that precedes the RPC call. |
| profiling | `\{ enabled?: boolean; includeServerBreakdown?: boolean; mode?: "summary" \| "verbose" \}` | ✗ | Per-call profiler configuration; when present, overrides the SDK's global profiler settings for this request. |
| timeout | `number` | ✗ | Per-call RPC timeout in milliseconds; overrides the SDK-level default for this request only. |

## Returns

`Promise<void>` — Resolves when the embeddings are deleted.

## Throws

| Error | When |
| --- | --- |
| `RAGDeleteFailedError` | When the operation fails or workspace doesn't exist |
| `RAG_DELETE_FAILED` | The delete operation fails or workspace doesn't exist |

## Example

```typescript
await ragDeleteEmbeddings({
  ids: ["doc-1", "doc-2"],
  workspace: "my-docs",
});
```
