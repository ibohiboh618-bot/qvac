---
title: "translate( )"
titleStyle: code
description: Translates text from one language to another using a specified translation model.
---

```ts
function translate(params: TranslateClientParams, options?: RPCOptions): { stats: Promise<{ cacheTokens?: number; decodeTime?: number; encodeTime?: number; timeToFirstToken?: number; tokensPerSecond?: number; totalTime?: number; totalTokens?: number } | undefined>; text: Promise<string>; tokenStream: AsyncGenerator<string> };
```

Supports both NMT (Neural Machine Translation) and LLM models.

## Parameters

| Name | Type | Required? | Description |
| --- | --- | :---: | --- |
| params | [`TranslateClientParams`](#translateclientparams) | ✓ | Translation configuration object |
| options | `RPCOptions` | ✗ | Optional RPC options (timeout, profiling, force new connection, etc.). |

### `TranslateClientParams`

| Field | Type | Required? | Description |
| --- | --- | :---: | --- |
| modelId | `string` | ✓ | — |
| text | `string \| string[]` | ✓ | — |
| stream | `boolean` | ✓ | — |
| modelType | `"llamacpp-completion" \| "nmtcpp-translation" \| "llm" \| "nmt"` | ✓ | — |

### `options`

| Field | Type | Required? | Description |
| --- | --- | :---: | --- |
| forceNewConnection | `boolean` | ✗ | When `true`, skip any cached RPC connection and open a fresh one for this call. |
| healthCheckTimeout | `number` | ✗ | Timeout in milliseconds for the health-check probe that precedes the RPC call. |
| profiling | `\{ enabled?: boolean; includeServerBreakdown?: boolean; mode?: "summary" \| "verbose" \}` | ✗ | Per-call profiler configuration; when present, overrides the SDK's global profiler settings for this request. |
| timeout | `number` | ✗ | Per-call RPC timeout in milliseconds; overrides the SDK-level default for this request only. |

## Returns

`object` — Object with the following fields:

| Field | Type | Description |
| --- | --- | --- |
| stats | `Promise<\{ cacheTokens?: number; decodeTime?: number; encodeTime?: number; timeToFirstToken?: number; tokensPerSecond?: number; totalTime?: number; totalTokens?: number \} \| undefined>` | Translation statistics |
| text | `Promise<string>` | Complete translated text |
| tokenStream | `AsyncGenerator<string>` | Stream of translated tokens |

### `stats`

| Field | Type | Required? | Description |
| --- | --- | :---: | --- |
| cacheTokens | `number` | ✗ | — |
| decodeTime | `number` | ✗ | — |
| encodeTime | `number` | ✗ | — |
| timeToFirstToken | `number` | ✗ | — |
| tokensPerSecond | `number` | ✗ | Tokens processed per second |
| totalTime | `number` | ✗ | Total embedding time in milliseconds |
| totalTokens | `number` | ✗ | Total tokens processed |

## Throws

| Error | When |
| --- | --- |
| `QvacErrorBase` | When translation fails with an error message or when language detection fails |
| `TRANSLATION_FAILED` | Translation fails |

## Example

```typescript
// Streaming mode (default)
const result = translate({
  modelId: "modelId",
  text: "Hello world",
  from: "en",
  to: "es"
  modelType: "llm",
});

for await (const token of result.tokenStream) {
  console.log(token);
}

// Non-streaming mode
const response = translate({
  modelId: "modelId",
  text: "Hello world",
  from: "en",
  to: "es"
  modelType: "llm",
  stream: false,
});

console.log(await response.text);
```
