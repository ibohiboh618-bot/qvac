---
title: "getLogger( )"
titleStyle: code
description: Creates or retrieves a namespaced logger instance.
---

```ts
function getLogger(namespace: string, options?: LoggerOptions): Logger;
```

## Parameters

| Name | Type | Required? | Description |
| --- | --- | :---: | --- |
| namespace | `string` | ✓ | Namespace used to prefix log messages from this logger (e.g. `"my-app"`, `"@qvac/sdk:embed"`). |
| options | [`LoggerOptions`](#loggeroptions) | ✗ | Optional logger configuration (custom transports, log level, etc.). When provided, a new logger is always constructed. |

### `LoggerOptions`

| Field | Type | Required? | Description |
| --- | --- | :---: | --- |
| level | `any` | ✗ | Minimum log level that will be emitted. |
| namespace | `string` | ✗ | Namespace prefix applied to every message emitted by this logger. |
| transports | `LogTransport[]` | ✗ | Additional log transports to receive every emitted message. |
| enableConsole | `boolean` | ✗ | When `false`, disables the default console transport. |

## Returns

[`Logger`](../shared-types#logger) — A logger instance.

## Example

```typescript
import { getLogger } from "@qvac/sdk";

const logger = getLogger("my-app");

logger.info("Application started");
logger.debug("Debug details:", { key: "value" });

logger.setLevel("error");
logger.info("This will not be logged");

const verboseLogger = getLogger("my-app:verbose", {
  level: "debug",
  enableConsole: true,
  transports: [(level, namespace, message) => {
    // Custom transport: write to file, send to server, etc.
  }],
});
```
