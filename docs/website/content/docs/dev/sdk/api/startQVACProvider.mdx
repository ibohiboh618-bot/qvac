---
title: "startQVACProvider( )"
titleStyle: code
description: Starts a provider service that offers QVAC capabilities to remote peers.
---

```ts
function startQVACProvider(params: ProvideParams): Promise<{ error?: string; publicKey?: string; success: boolean; type: "provide" }>;
```

The provider's keypair can be controlled via the seed option or the `QVAC_HYPERSWARM_SEED` environment variable.

## Parameters

| Name | Type | Required? | Description |
| --- | --- | :---: | --- |
| params | `ProvideParams` | ✓ | Options object with required topic, optional seed, and optional firewall config. |

### `params`

| Field | Type | Required? | Description |
| --- | --- | :---: | --- |
| firewall | `\{ mode: "allow" \| "deny"; publicKeys: string[] \}` | ✗ | Optional firewall configuration to allow/deny specific public keys. |
| topic | `string` | ✓ | Topic hex string for peer discovery. |

## Returns

`Promise<object>` — The provide response containing `success`, `publicKey`, and optional `error`.

| Field | Type | Description |
| --- | --- | --- |
| error | `string` | — |
| publicKey | `string` | — |
| success | `boolean` | — |
| type | `"provide"` | — |

## Throws

| Error | When |
| --- | --- |
| `QvacErrorBase` | When the response type is not "provide" or the request fails. |
| `INVALID_RESPONSE_TYPE` | Response type does not match expected `"provide"` |
| `PROVIDER_START_FAILED` | The server reports provider start failure |

## Example

```typescript
const response = await startQVACProvider({
  topic: "a1b2c3d4...",
  firewall: {
    mode: "allow",
    publicKeys: ["peer-public-key-hex"],
  },
});
console.log("Provider public key:", response.publicKey);
```
