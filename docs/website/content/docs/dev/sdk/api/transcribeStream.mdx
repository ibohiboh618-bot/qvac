---
title: "transcribeStream( )"
titleStyle: code
description: Streams audio transcription results in real-time, yielding text chunks as they become available from the model.
---

<Callout type="warn" title="Deprecated">
Pass audio via `transcribe()` instead. This overload will be
removed in the next major version.

Streaming transcription with upfront audio: sends full audio, yields text
chunks as they arrive.
</Callout>

```ts
function transcribeStream(params: TranscribeClientParams, options?: RPCOptions): AsyncGenerator<string>;
function transcribeStream(params: TranscribeStreamClientParams, options?: RPCOptions): Promise<TranscribeStreamSession>;
```

Yields text chunks as they become available from the model.

## Parameters

| Name | Type | Required? | Description |
| --- | --- | :---: | --- |
| params | [`TranscribeClientParams`](#transcribeclientparams) | ✓ | Transcription parameters including audio source. |
| options | `RPCOptions` | ✗ | Optional RPC options including per-call profiling. |

### `TranscribeClientParams`

| Field | Type | Required? | Description |
| --- | --- | :---: | --- |
| modelId | `string` | ✓ | — |
| audioChunk | `string \| Buffer<ArrayBufferLike>` | ✓ | — |
| prompt | `string` | ✗ | — |

### `options`

| Field | Type | Required? | Description |
| --- | --- | :---: | --- |
| forceNewConnection | `boolean` | ✗ | When `true`, skip any cached RPC connection and open a fresh one for this call. |
| healthCheckTimeout | `number` | ✗ | Timeout in milliseconds for the health-check probe that precedes the RPC call. |
| profiling | `\{ enabled?: boolean; includeServerBreakdown?: boolean; mode?: "summary" \| "verbose" \}` | ✗ | Per-call profiler configuration; when present, overrides the SDK's global profiler settings for this request. |
| timeout | `number` | ✗ | Per-call RPC timeout in milliseconds; overrides the SDK-level default for this request only. |

## Returns

`AsyncGenerator<string>` — Yields text chunks as they are transcribed.

## Throws

| Error | When |
| --- | --- |
| `TRANSCRIPTION_FAILED` | Transcription fails |

## Example

```typescript
const stream = transcribeStream({
  modelId: "whisper-model",
  audioChunk: "/path/to/audio.wav",
});

for await (const textChunk of stream) {
  process.stdout.write(textChunk);
}
```
