---
title: "getModelByName( )"
titleStyle: code
description: Looks up a model in the built-in catalog by its constant name.
---

```ts
function getModelByName(name: string): RegistryItem | undefined;
```

## Parameters

| Name | Type | Required? | Description |
| --- | --- | :---: | --- |
| name | `string` | ✓ | The model constant name (e.g. `"WHISPER_TINY"`) to look up. |

## Returns

[`RegistryItem`](../shared-types#registryitem) ` | undefined` — The matching model constant, or `undefined` if not found.

## Example

```typescript
import { getModelByName } from "@qvac/sdk";

const model = getModelByName("Llama 3.2 3B Q4");
if (model) {
  console.log(model.modelId, model.expectedSize);
}
```
